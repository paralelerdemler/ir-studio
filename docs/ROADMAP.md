# IR Studio Roadmap

IR Studio is an infrared command learning, testing, analysis, and export tool.

## v0.1 MVP

- [x] FastAPI backend
- [x] Broadlink RM4 Mini discovery
- [x] Broadlink IR learning
- [x] IR command sending
- [x] JSON command storage
- [x] React dashboard
- [x] Learn command from UI
- [x] List learned commands
- [x] Send learned command from UI
- [x] Delete learned command from UI
- [x] GitHub repository setup
